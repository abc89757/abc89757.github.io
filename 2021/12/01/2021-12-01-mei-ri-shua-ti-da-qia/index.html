剑指 Offer 09. 用两个栈实现队列

用两个栈实现一个队列。队列的声明如下，请实现它的两个函数 appendTail 和 deleteHead ，分别完成在队列尾部插入整数和在队列头部删除整数的功能。(若队列中没有元素，deleteHead 操作返回 -1 )

示例 1：

输入：
[“CQueue”,”appendTail”,”deleteHead”,”deleteHead”]
[[],[3],[],[]]
输出：[null,null,3,-1]

用两个栈模拟队列，看起来就和游戏王的魔法筒差不多,我们准备两个队列，一个sta1负责插入，一个sta2负责删除。

插入就直接把数字插入sta1即可，如果要删除，先看看sta2里有没有元素，如果没有就把sta1所有的元素都插入到sta2中，同时sta1要清空，这样在sta2的栈顶就是队列的头元素了（比如一开始插入顺序是1 2 3，此时3在sta1栈顶，队列头应该是1，我们把sta1的元素挪到sta2上，就是先挪3，再挪2，再挪1，这时在sta2里元素是3 2 1，1是sta2的栈顶元素，我们把1出栈即是删除了队列的头元素，删完后下一个元素2就是队列的下一个头元素）。要注意的是，在弹出sta2头顶元素前要判断sta2是否仍是空，如果还是空就返回-1，不为空就把栈顶元素返回并弹出。

class CQueue {
public:
    stack<int>sta1;
    stack<int>sta2;
    CQueue() {

    }
    
    void appendTail(int value) {
        sta1.push(value);
    }
    
    int deleteHead() {
        if(sta2.empty())
        {
            while(!sta1.empty())
            {
                sta2.push(sta1.top());
                sta1.pop();
            }
        }
        if(sta2.empty())return -1;
        int num=sta2.top();
        sta2.pop();
        return num;
    }
};

/**
 * Your CQueue object will be instantiated and called as such:
 * CQueue* obj = new CQueue();
 * obj->appendTail(value);
 * int param_2 = obj->deleteHead();
 */

剑指 Offer 30. 包含min函数的栈

定义栈的数据结构，请在该类型中实现一个能够得到栈的最小元素的 min 函数在该栈中，调用 min、push 及 pop 的时间复杂度都是 O(1)。

示例:

MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.min(); –> 返回 -3.
minStack.pop();
minStack.top(); –> 返回 0.
minStack.min(); –> 返回 -2.

准备两个栈拿来存数，一个sta是按照给的数来存数，一个min是按照升序序列来存，当往sta栈存数时判断一下这个数是否小于min栈的顶端元素或者min是否为空，如果为空或者数小于等于栈顶元素就把这个数同时存入min栈里。弹出元素时判断一下sta弹出的元素是否和min栈顶元素相同，如果相同就把min栈里的也弹出，当需要返回最小值时把min栈的顶端元素返回即可。

class CQueue {
public:
    stack<int>sta1;
    stack<int>sta2;
    CQueue() {

    }
    
    void appendTail(int value) {
        sta1.push(value);
    }
    
    int deleteHead() {
        if(sta2.empty())
        {
            while(!sta1.empty())
            {
                sta2.push(sta1.top());
                sta1.pop();
            }
        }
        if(sta2.empty())return -1;
        int num=sta2.top();
        sta2.pop();
        return num;
    }
};

/**
 * Your CQueue object will be instantiated and called as such:
 * CQueue* obj = new CQueue();
 * obj->appendTail(value);
 * int param_2 = obj->deleteHead();
 */

剑指 Offer 32 - III. 从上到下打印二叉树 III

请实现一个函数按照之字形顺序打印二叉树，即第一行按照从左到右的顺序打印，第二层按照从右到左的顺序打印，第三行再按照从左到右的顺序打印，其他行以此类推。

例如:
给定二叉树: [3,9,20,null,null,15,7],

  3
 / \
9  20
  /  \
 15   7
返回其层次遍历结果：

[
  [3],
  [20,9],
  [15,7]
]

层序遍历完后每隔一层把一层的节点翻转一下。

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>>v;
        if(!root)return v;
        queue<TreeNode*>que;
        que.push(root);
        while(!que.empty())
        {
            int len=que.size();
            vector<int>v1;
            for(int i=0;i<len;i++)
            {
                v1.push_back(que.front()->val);
                if(que.front()->left)que.push(que.front()->left);
                if(que.front()->right)que.push(que.front()->right); 
                que.pop();
            }
            v.push_back(v1);
        }
        for(int i=1;i<v.size();i+=2)
        {
            reverse(v[i].begin(),v[i].end());
        }
        return v;
    }
};
